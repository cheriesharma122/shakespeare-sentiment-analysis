# Sentiment Analysis of Shakespeare Text using NLP

## 📌 Introduction
This project performs sentiment analysis on text written by William Shakespeare using basic Natural Language Processing (NLP) techniques. The goal is to classify text as positive, negative, or neutral based on word usage.

## 🎯 Objective
- Understand basic NLP concepts  
- Perform sentiment analysis using Python  
- Classify text without using machine learning  

## ⚙️ Approach
1. Convert text to lowercase  
2. Remove punctuation  
3. Tokenize into words  
4. Compare words with predefined positive and negative lists  
5. Calculate sentiment score  
6. Classify as Positive, Negative, or Neutral  

## 🧠 Results
- Successfully analyzed multiple Shakespeare quotes  
- Generated sentiment scores for each quote  
- Displayed overall summary of sentiments  

## ⚠️ Difficulties Faced
- Understanding NLP without machine learning  
- Handling text preprocessing  
- Running code in terminal  
- Selecting correct sentiment words  

## 🛠️ Solutions
- Broke problem into smaller steps  
- Tested code multiple times  
- Improved word lists gradually  
- Debugged errors step-by-step
